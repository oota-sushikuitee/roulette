## Quick Start

Clone the repository

```bash
$ git clone https://github.com/oota-sushikuitee/roulette
```

Change directory

```bash
$ cd roulette
```

Install dependencies

```bash
$ yarn install
```

Start the application

```bash
$ yarn dev
```
